# EVM contracts (skeleton — flesh out in W8)

Solidity port of the Sui Move modules. Deployment target TBD in W8 (likely
Base or Arbitrum testnet first, then mainnet alongside the BSC chapter DLC).

## Modules

- `Chronicle.sol` — ERC-721, transferable. Mirrors `chronicle::chronicle` field
  by field.
- `WitnessSeal.sol` — ERC-5192 soulbound. `locked(tokenId)` is permanently
  `true`; transfers revert.

## Deps

Install OpenZeppelin (not committed):

```bash
forge install OpenZeppelin/openzeppelin-contracts --no-commit
```

Remappings live in `remappings.txt` (auto-generated by `forge remappings`).

## Build / test

```bash
forge build
forge test -vv
```
